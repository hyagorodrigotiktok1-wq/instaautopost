1. Conectar ao GitHub

O dashboard salva os posts no seu repositorio via GitHub API.

Crie em github.com/settings/tokens com permissao Contents: Read and write

2. Cloudinary (upload de videos)

Permite fazer upload de videos direto do seu computador. Gratuito ate 25GB.

1. Crie uma conta gratis em cloudinary.com
2. O Cloud Name aparece no Dashboard logo apos o login
3. Va em Settings > Upload > Upload presets e crie um preset com Signing Mode: Unsigned

3. Configurar o Instagram

Cada conta do Instagram precisa ser Business ou Creator e estar conectada a uma Pagina do Facebook.
1
Acesse developers.facebook.com e crie um App do tipo Business
2
No App, adicione o produto Instagram Graph API
3
Em Graph API Explorer, gere um token com as permissoes:
instagram_basic instagram_content_publish pages_read_engagement
4
Troque o token de curta duracao por um Long-Lived Token (dura 60 dias):
GET https://graph.facebook.com/v21.0/oauth/access_token?grant_type=fb_exchange_token&client_id={APP_ID}&client_secret={APP_SECRET}&fb_exchange_token={SHORT_TOKEN}
5
Descubra seu Instagram User ID:
GET https://graph.facebook.com/v21.0/me/accounts?access_token={TOKEN}
Pegue o id da Pagina, depois:
GET https://graph.facebook.com/v21.0/{PAGE_ID}?fields=instagram_business_account&access_token={TOKEN}
6
No seu repositorio GitHub, va em Settings > Secrets > Actions e adicione:
  • INSTAGRAM_ACCESS_TOKEN = seu token
  • INSTAGRAM_USER_ID = seu user ID
O token expira em 60 dias. Coloque um lembrete pra renovar! Futuramente podemos adicionar auto-refresh.

4. Contas extras (opcional)

Para mais contas, adicione secrets neste formato:

INSTA_ACCOUNT_MINHACONTA_TOKEN
INSTA_ACCOUNT_MINHACONTA_USER_ID

Depois selecione o nome da conta (ex: "minhaconta") ao agendar um post.

Isso atualiza o dropdown de contas no formulario de novo post.

Sobre

InstaAutoPost e uma ferramenta gratuita que agenda e publica Reels no Instagram automaticamente via GitHub Actions + Instagram Graph API.

Custo: R$ 0,00 (GitHub Actions gratuito para repositorios publicos).
O workflow roda a cada 30 minutos e publica os Reels agendados.
